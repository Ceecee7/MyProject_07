# MyProject_07

## 项目简介
一个轻量级的 AI 新闻热点分析工具，支持多平台监控、情感分析、自动推送，无需编程即可部署。

## 功能特性
- 多平台热点聚合（抖音、知乎、B 站、华尔街见闻等）
- AI 情感分析
- 自动推送（企业微信/飞书/钉钉/Telegram/邮件）
- 30 秒 Docker 部署
- 支持 MCP 协议，自然语言交互分析

## 快速开始
```bash
git clone https://github.com/Ceecee7/MyProject_07.git
cd MyProject_07
pip install -r requirements.txt
python main.py
