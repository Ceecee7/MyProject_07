# 项目设计文档

## 1. 项目背景
信息过载时代，用户需要快速、准确地掌握多平台热点舆情。

## 2. 项目目标
- 聚合 35+ 平台热点（抖音、知乎、B 站、华尔街见闻等）
- 实时 AI 情感分析
- 30 秒 Docker 一键部署
- 支持企业微信 / 飞书 / 钉钉 / Telegram / 邮件自动推送

## 3. 技术栈
- Python 3.10
- FastAPI：轻量级 Web 框架
- MCP 协议：自然语言交互
- Docker & Docker Compose：快速部署
- GitHub Actions：CI/CD（后续扩展）

## 4. 核心模块
| 模块        | 职责                         |
|-------------|------------------------------|
| hotspot     | 多平台热点抓取                |
| sentiment   | AI 情感分析                   |
| notify      | 多通道推送                    |
| mcp         | 自然语言查询与对话            |

见仓库主页 `README.md`「快速开始」一节。
